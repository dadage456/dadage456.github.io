ios 的文档管理： 分享、导入、导出保存

#iOS11 读取沙盒以外的文档笔记

[TOC]

一、操作文档的方式

这里看出第二种操作方式，要比第一种简化了一个步骤。

第一种：使用了分享文档，读取其他APP的文档。

第二种：直接打开其他APP的文档。

###二、文件操作的三个步骤

配置info.plist ——> 读取文件——>操作文件

  • 配置info.plist

    • 配置——app可打开的文件的类型
    • 配置——自定义的文件类型

  • 读取文档

    • 应用直接读取沙盒以外的文档
    • 通过其他应用的调用系统的分享组件，分享文档

  • 操作文档： 创建、打开、保存等操作

###三、直接读取沙盒以外的文档

  • UIDocumentPickerViewController

    指定操作文件的类型， 操作的方式：打开、导入、导出、移动

  • UIDocumentBrowserViewController

    创建基于文档视图的App, XCode ——> File ——> Project——> Document Based App

    使用回调来完成文档的创建、文档的选择操作。

    • 添加自定义的动作：设置customActions属性，在长按弹出菜单、导航栏菜单功能项增加菜单

    • 执行回调方法，在UIActivityController分享栏添加自定义动作

    • 使用UIDocumentBrowserTransitionController实现动画过渡

  • 对读取文档的操作

    • 使用NSFileCoordinator访问选中的文档

    • 使用UIDocument对文档进行创建、打开、保存、重命名、定时保存等操作。

四、如何让应用在Files中显示Document文档中的内容

  • 应用如何显示在Files的设置

    • 在info.plist文件设置

      • Application supports iTunes file sharing: true

        在iTunes中显示应用的Documents文件夹

      • Supports opening documents in place: true

        在Files中显示该应用的Documents文件夹

    • 针对与基于文档浏览器UIDocumentBrowserViewController，在info.plish文件中设置

      • Supports Document Browser: true

        Files中是否可以直接点击文件，使用该应用打开。

  • 在iOS的设置——》隐私——》文件和文件夹, 设置当前应用允许显示。

  • 如果要隐藏Documents文档中的文档，文件名前加上.

  • 设置自定义文件的缩略图： 缩略图扩展 Thumbnail Extension
  • 设置自定义文件的快速预览： 快速预览扩展 Quick Look Preview Extension
