为自己的库创建Pod

发表于 2017-02-10

使用Cocoapods可以很方便的管理第三方类库，我们自己在开发过程中也会总结出自己常用的一些类库。我们也想让我们的类库用Cocoapods管理。

使用Cocoapods可以创建私有的Pod用于公司内容，也可以提交到Cocoapods truck.

  • 创建私有的Pod，用于公司内部。(官方教程)

  • 提交自己项目的podspec文件到CocoaPods trunk。官方提供教程

其实官方给的教程就很全了，没必要看这个文档。

创建项目的podspec文件

不管创建私有的Pod、还是发布到trunk,第一步就是要创建podspec文件。

  • podspec文件的作用
    告诉cocoapods关于类库的基本信息、源码地址、资源内容等。([具体介绍podSpec参数])

  • podspec文件的创建与验证

    • pod lib create : 自动生成一个pod项目所需的所有文件。包括类podSpec文件、DEMO工程。
    • pod lib lint : 验证podSpec文件是否正确。
    • pod spec create your_pod_spec_name : 手动创建podSpec文件

创建私有的Pod

  • 用于公共类库DEMO工程使用

  • 直接指定一个podspec文件、或包含podspec文件的git源

提交podspec文件到 Cocoapods trunk

前提：已经为自己的类库创建了podspec文件，并通过验证。

  • 注册Cocoapods trunk邮箱
